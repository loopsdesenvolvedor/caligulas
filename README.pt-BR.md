# Caligula's

Um projeto responsivo **Next.js + TypeScript + Redux** com um header modular, botões interativos e componentes reutilizáveis.

## Funcionalidades

- Header responsivo para desktop e mobile
- Botões interativos: menu, pesquisa, favoritos, usuário
- Gerenciamento global de estado com Redux Toolkit
- Estrutura modular de pastas: `ui` para componentes de funcionalidades, `shared` para componentes reutilizáveis
- TypeScript para tipagem e segurança

## Estrutura de Pastas

```text
src/
├─ ui/header/     # Componentes específicos do header
├─ shared/        # Componentes reutilizáveis (ButtonWithIcon, etc.)
├─ store/         # Slices e store do Redux
└─ styles/        # Estilos globais e configuração do Tailwind
```
